하단부 계획

search-result component와 유사하므로, search-document-list component을 새로 생성해서 리스트 주루룩 나오는 부분 분류. 주제 버튼 누르면 제목 뿐만이 아니라 그냥 search query을 보내도록...

고려해볼 수 있는점: 현재 주제 분류 레퍼런스 테이블 결과 = [주제 : 정치, 문서 : [문서1 id, 문서2 id, ...]] 그러면 해당 주제를 선택하면, 레퍼런스 테이블에서 문서 id을 불러와서 ES에 쿼리 요청해서 리스트 불러옴. 10개 단위로 끊는다고 하면 limit 쿼리 사용 가능해보인다.
